\chapter{引言}\label{chap:introduction}

动态目标抓取是机器人操作中兼具感知、预测和闭环控制难度的典型问题。机器人抓取研究长期经历了从解析几何、接触约束和运动规划到数据驱动抓取合成的转变，近期综述也指出，复杂非结构化场景下的鲁棒抓取通常需要同时处理感知、规划和控制的不确定性\citep{zhang2022robotic,newbury2023deep}。与静态抓取相比，动态抓取要求机械臂在目标持续运动的过程中完成接近、相对位姿对齐、夹爪闭合以及后续运输；目标位置、姿态和速度的变化会直接影响可行抓取窗口，观测噪声和时延则进一步增加闭环决策的不确定性。因此，动态抓取不仅考验单步轨迹跟踪能力，也要求策略在有限时间内对目标运动趋势和任务阶段进行稳定判断。

基于模型的控制方法在机器人任务中具有清晰的物理结构和较强的可解释性。模型预测控制（model predictive control, MPC）能够在滚动时域内显式处理位置、姿态、平滑性和执行约束，在状态准确时通常表现稳定。然而，当目标位姿估计含有高频扰动或时延时，确定性参考构造可能将观测误差直接传递到末端轨迹，使闭爪时机、接触姿态或优化求解发生退化。模仿学习提供了另一条路径：策略从专家示教中学习观测到动作的映射，并可通过历史窗口和动作块输出在一定程度上吸收短时噪声与运动趋势。但模仿学习也面临分布偏移、示教覆盖不足以及闭环误差累积等问题。

近年来，机器人模仿学习从简单行为克隆逐步发展到序列模型、动作分块模型和生成式策略。行为克隆把专家轨迹转化为监督学习样本，其思想可追溯到端到端驾驶策略ALVINN\citep{pomerleau1989alvinn}，但离线监督策略在闭环执行中会受到分布偏移和误差累积影响\citep{ross2010efficient,ross2011reduction}。循环模型通过隐状态编码历史观测，有利于动态任务中的速度估计和阶段识别\citep{mandlekar2021matters}；Action Chunking Transformer（ACT）类方法一次预测未来动作块，以降低逐步控制误差\citep{zhao2023learning}；扩散策略通过条件去噪过程建模动作分布，并在机器人操作任务中显示出生成平滑动作序列的潜力\citep{ho2020denoising,song2020denoising,chi2023diffusion}。这些方法在模型容量、时间建模和动作分布表达上具有不同归纳偏置，适合在统一表示和统一执行器下进行比较。

本文研究仿真环境中的动态立方体抓取问题，关注目标中心化模仿策略在真值观测和噪声观测下的闭环表现。算法专家在真值状态下结合有限状态机和MPC生成示教；学习策略接收历史观测窗口和未来目标条件，输出一段任务空间参考动作，再由共享MPC执行器转化为机器人控制。该设计将学习问题限制在动态拦截、相对位姿对齐和闭爪时机预测上，同时保留下层模型控制对执行约束的处理能力。本文比较行为克隆（BC）、循环行为克隆（BC-RNN）、ACT和条件扩散策略四类模仿学习模型，并以MPC专家作为模型控制参考。

本文的主要贡献如下：
\begin{enumerate}
    \item 提出一种用于动态立方体抓取的目标中心化模仿学习框架，将历史相对状态、未来目标条件和相对任务空间动作统一到同一坐标表示中，降低全局平移和自旋对监督学习目标的影响。
    \item 在相同数据表示、相同动作块接口和相同MPC执行器下比较BC、BC-RNN、ACT和条件扩散策略，分析不同模型结构对闭环抓取成功率和轨迹质量的影响。
    \item 构建真值观测与多类噪声观测评估矩阵，定量比较学习策略与MPC专家在运动分解场景和感知退化场景中的性能差异。
    \item 通过成功率、Wilson置信区间、最终距离、跟踪误差和轨迹平滑度等指标，讨论动态抓取中失败模式、噪声敏感性和动作块策略的实际作用。
\end{enumerate}

本文结构安排如下。第\ref{chap:related-work}章回顾模仿学习、生成式动作策略、模型预测控制和动态目标抓取相关研究。第\ref{chap:method}章给出任务定义、专家生成、目标中心化表示、模型结构和评估指标。第\ref{chap:results}章报告实验结果并分析不同策略与噪声条件下的性能差异。第\ref{chap:conclusion}章总结全文并讨论进一步提升动态抓取鲁棒性的方向。
